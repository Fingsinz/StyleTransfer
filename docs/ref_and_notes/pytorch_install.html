PyTorch 安装

2025-02-07

Reference: Zero to Mastery Learn PyTorch for Deep Learning

PyTorch 基本环境搭建

  1. 创建并激活环境
python -m venv [venv-name]
[venv-name]\Scripts\activate
  1. 安装 Pytorch
pip install torch torchvision torchaudio

验证 Pytorch 安装，出现版本号则为正常。

PyTorch-GPU 环境搭建

在搭建虚拟环境后，如果需要在 GPU 上运行，需要安装 PyTorch-GPU 版本。

  1. 确定自己的 GPU CUDA 版本。
nvidia-smi
  1. 下载对应的 PyTorch-GPU 版本。官方引导下载
